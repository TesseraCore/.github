# TesseraCore · Org-wide Community Files

This repository provides **default community health files** for the TesseraCore organization:
- Code of Conduct
- Contributing Guide
- Security Policy
- Support Guide
- Issue / PR templates
- Dependabot & reusable CI

These defaults apply across org repos unless overridden locally.

**What is TesseraCore?**  
A universal, framework-agnostic **deep-zoom engine** for multi-layer imagery: color-accurate tiling, high-precision annotations, plugin architecture, and graceful fallback (**WebGPU → WebGL2 → WebGL → Canvas**).
